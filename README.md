# Smart Dustbin -- Android App

A simple IoT-based smart trash bin that utilizes a sensors to measure data and integrate with real-time services for online updates. Designed as a beginner-friendly project to explore the intersection of hardware, mobile cloud integration, and Internet of Things (IoT).

<details open>

<summary>Technology Stack</summary>

|    Component    | Description              |
|:---------------:|:------------------------ |
| Microcontroller | ESP32 Dev Board          |
|      Cloud      | Firebase (Realtime, FCM) |
|   Mobile App    | Java/Kotlin                         |

</details>

## Mobile Cloud Integration

This project explores **Mobile Cloud Computing** by:
- Sending bin usage data to the cloud
- Allowing remote control via mobile app
- Triggering notifications when bin is full
